분석 방법

ENJOY 테스트가 동작하는 방식

처리 흐름

  1. 업로드된 얼굴 이미지를 분석 가능한 표준 크기로 정규화
  2. 얼굴 영역 중심으로 특징 벡터 추출
  3. 내부 이름 아키타입 기준과의 유사도 계산
  4. Top3 결과와 신뢰도 배지 생성

모델 성격

본 서비스는 고정밀 신원인증 모델이 아니라, 오락 목적의 경량 인상 매칭 파이프라인입니다. 결과는 사용자 경험과 재미를 위한 표현이며, 환경(조명/각도/화질)에 따라 변동될 수 있습니다.

중요한 한계와 고지

결과는 이름에서 연상되는 외적 분위기를 LLM이 해석한 인상 매칭이며, 실제 사람의 정체성/능력을 판정하지 않습니다. 또한 실제 나는솔로 출연자 사진을 학습한 식별 모델이 아닙니다.

품질 개선 방식

ENJOY는 사용자 피드백을 바탕으로 UI/UX, 촬영 가이드, 실패 케이스 처리, 테스트 자동화 게이트를 지속적으로 개선합니다. 기능 변경 시 lint/test/build/e2e 게이트를 통과한 뒤 배포합니다.
